# Mobile Pong Game

A responsive, mobile-friendly Pong game built with HTML5 Canvas and JavaScript.

## Features

- 🎮 **Mobile Optimized** - Works perfectly on smartphones and tablets
- 🖱️ **Mouse & Touch Support** - Control with mouse movement or touch
- 🤖 **AI Opponent** - Play against an intelligent computer opponent
- 📊 **Score Tracking** - Keep track of both player and computer scores
- 🎨 **Modern UI** - Beautiful gradient background and glowing effects
- ⚡ **Smooth Gameplay** - 60 FPS gaming experience
- 📱 **Responsive Design** - Adapts to all screen sizes

## How to Play

1. **Open** `index.html` in your web browser
2. **Move** your mouse (desktop) or finger (mobile) to control the left paddle
3. **Beat** the computer opponent by hitting the ball back
4. **Score points** when the computer misses the ball
5. **Compete** for the highest score!

## Controls

- **Desktop**: Move your mouse up and down
- **Mobile**: Swipe your finger up and down on the screen

## Files

- `index.html` - Main HTML file with structure
- `style.css` - Styling and responsive design
- `game.js` - Game logic and mechanics

## Installation

1. Clone or download this repository
2. Place all three files (`index.html`, `style.css`, `game.js`) in the same directory
3. Open `index.html` in any modern web browser
4. Start playing!

## Game Mechanics

- The ball bounces off top and bottom walls
- Hit the ball with your paddle to keep it in play
- Computer has mild AI - it doesn't win instantly!
- Ball speed increases with intensity
- Ball trajectory is affected by where it hits the paddle

## Browser Compatibility

Works on all modern browsers:
- Chrome/Edge
- Firefox
- Safari
- Mobile browsers (iOS Safari, Chrome Mobile, etc.)
